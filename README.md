# Jean Fonseca Portfolio

Welcome to my professional portfolio! This site is built with [Astro](https://astro.build/) and is automatically deployed to GitHub Pages.

## 🚀 Built With

- **Astro**: The web framework for content-driven websites.
- **GitHub Actions**: Automated CI/CD for seamless deployment.
- **Vanilla CSS**: Modern, responsive design without the bloat.

## 🛠️ Getting Started

To run this project locally:

```sh
npm install
npm run dev
```

Happy coding!
